# DDoS工具
## 使用前必读
1. 安装必要库
```sh
pip install asyncio
pip install aiohttp
```
2. 下载或克隆存储库  
3. 用命令行`cd`到这个目录下
```sh
cd DDoS-Tool
```
4. 用Python运行
```sh
python -u main.py
```
5. 输入要攻击的目标URL
```url
www.example.com
```
6. 按`Enter`然后给自己倒杯茶
